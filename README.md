# 🌴 AI Travel Recommendation System

An AI-powered travel recommendation web application built with Flask, designed to help users discover the best destinations, plan trips, and explore Sri Lanka's most beautiful locations.

## ✨ Features

- 🏝️ Destination recommendations based on user preferences
- 📅 Trip planner
- 🍛 Local food guide
- ⭐ Rate & review destinations
- 🔥 Trending destinations
- 🔐 User login system
- 🛠️ Admin feedback panel

## 🛠️ Tech Stack

- **Backend:** Python (Flask)
- **Database:** SQLite (`destination.db`, `ceylon_vibe.db`)
- **Frontend:** HTML, CSS
- **Data:** Cleaned Sri Lanka travel dataset (CSV)
